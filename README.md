# S&P 500 Stock Market ML Prediction

A machine learning pipeline that predicts next-day stock direction (UP / DOWN) for 10 S&P 500 stocks using technical indicators and ensemble classifiers.

## Usage

**Step 1 -- Collect data:** Run `collect_data.ipynb`

Downloads 6 years of daily OHLCV data (Jan 2020 - Jun 2026) for 10 S&P 500 stocks via yfinance and saves `sp500_stocks.csv`.

**Step 2 -- Run ML pipeline:** Run `stock_ml_pipeline.ipynb`

Reads `sp500_stocks.csv`, engineers 20+ technical features, trains 3 models, evaluates them, generates charts, and saves trained models.

## Tech Stack

- **Python** -- pandas, numpy, scikit-learn, XGBoost, matplotlib, seaborn, yfinance
- **ML Models** -- Logistic Regression, Random Forest, XGBoost
- **Features** -- SMA, EMA, RSI, MACD, Bollinger Bands, momentum, volatility, lagged returns

## Feature Engineering

| Category | Features |
|----------|----------|
| Returns | daily return, log return |
| Trend | SMA (20, 50), EMA (12, 26), MACD + signal line |
| Momentum | RSI (14), 3-day and 5-day momentum |
| Volatility | Bollinger Bands, 5-day and 10-day rolling volatility |
| Volume | volume % change |
| Price Action | high-low range %, open-close change % |
| Lags | lagged close price and lagged returns (1, 3, 5 days) |

## Generated Charts

After running the pipeline, the following charts are saved in `images/`:

- **Model Comparison** -- side-by-side bar chart of all metrics
- **Confusion Matrices** -- per-model confusion heatmaps
- **ROC Curve** -- overlayed ROC curves with AUC scores
- **Precision-Recall Curve** -- per-model PR curves
- **Feature Importance** -- top 20 features for tree-based models
- **Correlation Heatmap** -- feature correlation matrix
- **Stock Technical Charts** -- price + SMA, RSI, MACD, Bollinger Bands for first 3 tickers
- **Class Distribution** -- target variable balance
